# 🍪 Idle Cookie Game (JavaScript)

A browser-based idle game built with HTML, CSS, and JavaScript featuring upgrade systems, achievements, prestige mechanics, and offline progress.

---

## 🎮 Live Demo

👉 https://tristan-com.github.io/idle-game/

---

## ✨ Features

* 🍪 Click-based resource generation
* ⚙️ Multiple upgrade tiers (Auto Clicker, Grandma, Farm, Factory)
* 📈 Real-time cookies-per-second system
* 💾 Persistent save system using localStorage
* ⏳ Offline progress calculation
* 🏆 Achievement system with unlockable milestones
* 🔁 Prestige system with permanent production bonuses
* 🎨 Responsive UI with animations and visual feedback
* 📊 Statistics tracking (total cookies, clicks, production)

---

## 🧠 How It Works

* Clicking the cookie increases your resources
* Upgrades generate cookies automatically over time
* Costs scale exponentially to balance progression
* Prestige resets progress but provides permanent bonuses
* Offline progress rewards players based on time away
* Achievements unlock based on gameplay milestones

---

## 📊 Example Gameplay Loop

1. Click to earn cookies
2. Buy upgrades to increase production
3. Unlock achievements
4. Prestige to gain permanent bonuses
5. Repeat with faster progression
